# 🧼 Window Cleaner (Unity Project)

## 📌 Overview

This project is a simple **Window Cleaning Simulation** built using Unity.
The player controls a sponge (sphere) using the mouse to clean dirt from a window surface.

---

## 🎮 Features

* 🖱 Mouse-based movement system
* 🧱 Procedural dirt generation (grid-based tiles)
* 🧼 Cleaning mechanic (tiles disappear on contact)
* 🎨 Realistic dirt texture applied
* 🖥 Basic UI with "Left Hand" and "Right Hand" options

---

## 🛠 Technologies Used

* Unity (2022+)
* C#
* Unity Physics System

---

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/YOUR_USERNAME/WindowCleaner.git
   ```
2. Open the project in Unity Hub
3. Open `SampleScene`
4. Press ▶ Play

---

## 📂 Project Structure

* `Assets/` → Scripts, Materials, Scenes
* `Packages/` → Unity dependencies
* `ProjectSettings/` → Project configuration

---

## 🎯 Gameplay

* Move the sponge using your mouse
* Clean all the dirt tiles from the window
* Tiles disappear as you clean them
